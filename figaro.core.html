figaro.core

A sweet Clojure API for Copycat

cas!

(cas! atom expected updated)

Atomically compares and sets the updated value when the expected value matches the current value of the atom.

client

(client nodes)

Returns a CopycatClient for the nodes. nodes should be a seq of maps containing :id, :host and :port values.

close!

(close! copycat)

Closes the copycat client or server.

dist-atom

(dist-atom client path)

Creates a distributed atom for the client on the path.

get

(get atom)

Gets a value from the atom.

mem-log

(mem-log)

Returns an in memory log.

server

(server log id port remote-nodes)

Returns a CopycatServer for the server id, port and remote-nodes. remote-nodes should be a seq of maps containing :id, :host and :port values.

set!

(set! atom value)

Sets the value for the atom.
